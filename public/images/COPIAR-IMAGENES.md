# Cómo usar tus imágenes

**Opción recomendada:** ejecuta el script desde la raíz del proyecto (PowerShell):

```powershell
.\scripts\copy-imagenes.ps1
```

El script copia desde **C:\Users\pablo.garcia\Pictures\Screenshots** a `public/images/` y `public/images/gallery/` con los nombres que usa la web.
